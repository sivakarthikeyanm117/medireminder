# MediReminder

MediReminder is a simple medicine reminder app. It is made to help elderly users who often forget to take their medicines on time. The app sends WhatsApp reminders for each scheduled medicine so users can take them regularly.


## About
This project was built mainly using **Vibe Coding (Cursor AI)**.  
The design was created in **Figma**, and the reminders are sent using **PyWhatKit** in Python.



## Features
- Add medicine schedules with time and dosage.  
- Get WhatsApp reminders automatically.  
- Easy to use for elderly users.  

## How it works
1. You set the medicine name and time.  
2. The app uses PyWhatKit to send a WhatsApp message at the right time.  
3. The reminder helps the user take medicine without missing doses.  

## Tools Used
- **Vibe Coding (Cursor AI)** → for building the app.  
- **Figma** → for design.  
- **PyWhatKit** → for WhatsApp automation.
